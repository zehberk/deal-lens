# Recorded Hyundai IONIQ 5 API fixtures

These fixtures were recorded from the Visor Public API on 2026-07-23 for active
2024–2026 Hyundai IONIQ 5 listings priced at or below $55,000, sorted newest
first. The listing pages contain 100 and 50 records. The Level 1 fixture contains
the complete per-year and per-trim facet plan used by the market pipeline.

The public response bodies are preserved as captured. API credentials,
authorization data, response headers, and usage/cost headers are excluded. Normal
pytest runs replay these files and must not contact the live API.
